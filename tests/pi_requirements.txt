# System Requirements:
# -------------------
# 1. Raspberry Pi OS (Raspbian Buster or later)
# 2. Python 3.9+
# 3. Enable SPI interface: sudo raspi-config -> Interface Options -> SPI -> Enable
# 4. Enable I2C interface: sudo raspi-config -> Interface Options -> I2C -> Enable
# 5. ADB tools: sudo apt-get install android-tools-adb
# 6. Camera permissions: Add user to video group: sudo usermod -a -G video $USER

# Hardware Connections:
# --------------------
# RFID MFRC522:
#   - SDA  -> GPIO 8 (CE0)
#   - SCK  -> GPIO 11 (SCKL)
#   - MOSI -> GPIO 10 (MOSI)
#   - MISO -> GPIO 9 (MISO)
#   - GND  -> GND
#   - RST  -> GPIO 25
#   - 3.3V -> 3.3V
#
# LEDs:
#   - Scan LED -> GPIO 22
#   - Binary Display (3-bit):
#     - BIN0 -> GPIO 23
#     - BIN1 -> GPIO 24
#     - BIN2 -> GPIO 25
